# 🏆 KBC-Inspired Flask Web App

A modern, interactive registration and data management system that replaces traditional JavaScript with **PyScript** for client-side logic.

## ✨ Features
- **Smart Registration:** Real-time username availability checker.
- **Python Frontend:** Uses PyScript to handle UI interactions and navigation.
- **Data Persistence:** SQLite integration via SQLAlchemy.
- **Dynamic UI:** Responsive design with custom toast notifications.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3, PyScript (WebAssembly)
- **Database:** SQLAlchemy (SQLite)
